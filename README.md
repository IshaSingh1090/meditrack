# 💊 MediTrack Pro

A full-stack Medicine Inventory Management System to track medicines, expiry dates, and stock levels.

## 🌐 Live Demo
👉 [meditrack-mauve-delta.vercel.app](https://meditrack-mauve-delta.vercel.app)

## ✨ Features
- 🔐 User Authentication (Register/Login)
- ➕ Add, Edit, Delete medicines
- 📊 Dashboard with Total, Expiring Soon, Expired stats
- 🔍 Search and filter medicines
- 📷 Barcode scanner support
- 🟢 Status badges (Safe / Expiring Soon / Expired)

## 🛠️ Tech Stack
| Frontend | Backend | Database | Deployment |
|----------|---------|----------|------------|
| React.js | Node.js | MongoDB | Vercel |
| Axios | Express.js | Mongoose | Render |
| CSS3 | JWT Auth | | GitHub Actions |

## 🚀 Getting Started

### Clone the repo
```bash
git clone https://github.com/IshaSingh1090/meditrack.git
cd meditrack
```

### Run Frontend
```bash
cd frontend
npm install
npm start
```

### Run Backend
```bash
cd backend
npm install
npm start
```
